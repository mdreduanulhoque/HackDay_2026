# Story Games - HackDay Project

## üéØ Project Mission

**HackDay Theme:** "Communication is Not a Barrier"

This project was developed for a HackDay with a focus on supporting children whose brain development differs from typical patterns, particularly those who face challenges with communication. Through engaging, story-based cognitive games, we aim to provide an accessible platform that helps these children develop essential cognitive skills in a fun, non-verbal way.

### Our Focus
Traditional communication can be a barrier for many children with developmental differences. This game platform removes that barrier by using:
- **Visual pattern recognition** instead of verbal instructions
- **Shape-based puzzles** that transcend language
- **Progressive difficulty** that adapts to individual learning pace
- **Positive reinforcement** through celebrations and encouraging feedback
- **Non-judgmental learning** where mistakes are opportunities to learn

By focusing on cognitive development through play, we help children build confidence and essential thinking skills without the pressure of verbal communication.

## üéØ Games Included

### Game 1: Pattern Recognition
Test your pattern recognition skills with shape-based puzzles.

### Game 2: Logic Challenges
Solve challenging logic puzzles to advance through levels.

### Game 3: Gene World Escape (Pattern Master)
Help the hero escape by solving 23 levels of increasingly difficult pattern puzzles:
- **Levels 1-8:** Sequence Patterns (recognize repeating shapes)
- **Levels 9-16:** Shift Patterns (identify row-based shifts)
- **Levels 17-23:** Matrix Patterns (Latin square logic)

**Scoring System:**
- Correct answer: +3 points
- Wrong answer: -1 point (minimum 0)

**Features:**
- 8 different shape types
- Confetti celebrations on correct answers
- Helpful hints from Gene on wrong answers
- Progress saving with localStorage
